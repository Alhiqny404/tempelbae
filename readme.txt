## Introduction
Tempelbin merupakan website tempat pengguna menulis, mengunggah, dan berbagi teks secara online, baik berupa teks biasa maupun berbagai script code pemrograman berbasis teks.
Website ini dibuat menggunakan teknologi PHP dengan konsep arsitektur MVC (untuk backend), Template Frogetor (untuk Frontend).


## made using
-- Device : Redmi 8
-- Tecnology programming : HTML, CSS, JAVASCRIPT, PHP
-- Database : MYSQLI
-- Code Editor : Acode Premium
-- Web Server : KSWEB Premium
-- Browser : Kiwi Browser


## Installasi
1. Extrak file dan Simpan Script di localhost
2. Buat database dan Import file db (db/tempelbin.sql)
3. Set Konfigurasi base url dan database (app/config/config.php)
4. Run code
